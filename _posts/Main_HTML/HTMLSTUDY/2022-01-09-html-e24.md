- 스타일 외부 파일에 둬야한다.

css 폴더를 따로 두고 그밑에  style.css 파일을 만들어서 list안에있던 style 들을 옮긴다.


그리고 head 안에다 
    <link href ="../css/style.css" type= "text/css" rel="stylesheet" >
를 링크해서 파일을 쓸수있게해준다.
