- HTML 문서의 기본 구조 :규칙

```HTML
<html>
    <head>
        <title> first pasge</title>
    </head>
    <body>
        <h1> Hello! HTML </h1>
        <p>
            <a href ="">HTML</a>이란
            HyperText Markup Language.
        </p>
    </body>
</html>
```

- head : 브라우저 화면에 보여줘야할 내용에 대한 설정
- title : 문서 제목
- body : 출력되야할 내용
- h1 : 본문 헤더
- p : 문장
