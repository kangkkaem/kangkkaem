- felxt-grow / flex-shrink / flex-basis의 축약표현

- 축약표현
flex-grow: 0;   
flex-shrink: 0;
flex-basis : 100px;f
--> flex : 1 1 100px    ( 순서가 중요하다.)

- 축약 값
flex : none;    -> flex : 0 0 auto;
flex : auto;    -> flex : 1 1 auto;
flex : initial  -> flex : 0 1 auto;
flex : 100px    -> flex : 1 1 100px;
