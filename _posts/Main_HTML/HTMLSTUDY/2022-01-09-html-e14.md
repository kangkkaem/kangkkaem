- 아웃라인 영역 비효율적인부분을 해결하기위해 시멘틱 태그가 등장했다.

- 시맨틱태그란 의미를 부여하고있는 태그이다.

제목은 제목으로 존재해야하기때문에 다시 제목을 원래대로 돌린다.

- 콘텐츠를 구분해줄 수 있게 의미가 살아있는 태그
    <div id ="header"/>
    <div id ="main" />
    <div id ="footer" />
    -> 사람들이 가장 많이 사용하고 가장 많이 추천하는 이름으로 표준화가 진행됐다.


- 문서를 만들때 통일감을 줄수있다.
```html
    <h1>
    <header><h2></header>
        <div><h2></div>
    <main><h2></main>
    <footer><h2></footer>
```
