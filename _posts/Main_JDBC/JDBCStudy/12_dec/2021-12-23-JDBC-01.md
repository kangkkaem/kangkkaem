JDBC ( JAVA DATABASE CONNETED )


- 사용자 요구에 부응한 쿼리를 만들어서 실행할수 있게 해주는것이다.
- 연결 API 가 차이가 있기 때문에 프로그램만드는데있어서 편리하기 위함이다.
- 오라클에 접속하기 위한 도구들을 어플리케이션을 만드는 사용자들이 직접 쓰지않게 하기위함 (단일화) - > JDBC라한다.
