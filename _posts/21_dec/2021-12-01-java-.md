1+(-2)+3+(-4)+... 으로 계속더해갔을때 몇까지 더해야 총합이 200 이상인가

```java
public class prac {

	public static void main(String[] args) {
		int sum = 0;
		int s = 1;
		int num = 0;
		
		for(int i=1;true; i++, s=-s) {
			num = s *i;
			sum +=num;
			
			if(sum>=200)
				break;
		}
		
		System.out.println("num="+num);
		System.out.println("sum="+sum);
	}
}
```
