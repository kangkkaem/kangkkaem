- new file을 눌러 Dynamic Web Project 파일을 생성한다.

![web](/assets/images/web.JPG)

- webContent 폴더아래 index.html 을 생성한다.

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="EUC-KR">
<title>Insert title here</title>
</head>
<body>
	환영합니다.
</body>
</html> 
```
-> 간단한 html문 작성


- ctrl + F11  -> tomcat 버전을 선택한다. (서버설정)

- 포트충돌나는것은 다른 포트를 쓰고잇기떄문에 쓰고있던 서버를 닫아주면된다.
